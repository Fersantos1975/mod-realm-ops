# RealmOps v0.2.0 RC1 installation

## Server module

1. Back up the current repository.
2. Replace the repository contents with this package while preserving `.git`.
3. Configure, build, and install AzerothCore.
4. Do not install if compilation fails.

```bash
cd /home/cura/azerothcore
cmake -S . -B build -DCMAKE_INSTALL_PREFIX=/home/cura/server -DMODULES=static
cmake --build build -j"$(nproc)"
cmake --install build
```

## WoW addon

Copy `addon/RealmOps` to:

```text
World of Warcraft/Interface/AddOns/RealmOps
```

Delete or replace the old RealmOps folder so stale files are not retained.
Restart WoW or use `/reload` after the server is running.

## Validation

```text
.realmops version
.realmops quest info 24499
.realmops quest audit 24499
.realmops instance audit 658 0
```

Check the following in the addon:

- Compatibility status and Release Supported text
- RealmOps icon on the main, floating, and minimap buttons
- Quest Audit Group and Export buttons
- Quest details and chain/audit scrollbars
- Select text with the mouse; use Ctrl+C to copy
- Instance report scrolling, text selection, filtering, and export

This release candidate was prepared from the uploaded source but must be compiled and tested against the target AzerothCore checkout before merging to `main`.
