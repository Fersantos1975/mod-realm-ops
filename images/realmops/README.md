# RealmOps Branding Assets

Official descriptor:

> **Operational Diagnostics Platform for AzerothCore**

Official motto:

> **Inspect. Diagnose. Resolve.**

The original supplied artwork is preserved as `realmops-master.png`.

PNG files are intended for GitHub, documentation, releases, and general use.
TGA files under `addon/RealmOps/Media/Textures/` are intended for the WoW 3.3.5a addon.

The icon crops retain the original dark background because automatic transparency removal would damage the detailed gear, shadows, glass, and metallic edges. A manually masked transparent master can be added later without altering these preserved assets.
