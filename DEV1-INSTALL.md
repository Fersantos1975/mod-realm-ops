# RealmOps 0.3.0-dev1 installation

This is a development checkpoint, not the stable 0.3.0 release.

## Module
Replace the repository files on your development branch, then configure/build/install AzerothCore as usual.

## AddOn
Copy `addon/RealmOps` to `World of Warcraft/Interface/AddOns/RealmOps`, replacing the previous RealmOps folder.

## Test
1. Start worldserver and log in with a GM account.
2. Open RealmOps and select Eligibility.
3. Search an instance, select a map, set difficulty 0-3, and run Group Audit.
4. Select a blocked player and use Open Quest to load the blocking quest chain.
5. Confirm legacy Quest and Instances tabs still work.
